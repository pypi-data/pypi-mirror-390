1.  Go to **CRM > Salesperson Planner > My Visits** or **All Visits**.
2.  Create a new visit.
3.  Or go to **CRM > Salesperson Planner > Visit Templates** and create a new recurring template to create periodical visits.
4.  For templates, select a Start Date.
5.  Set either a total number of repetitions or an End Date to calculate the last repetition.
6.  To reschedule a validated visit, either:
	- Change the date from the visit record.
	- Change the date from the calendar event automatically created for the visit.
